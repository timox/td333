**Entrées (16 au total côté DAW)
Inputs 1 à 4 — Combo XLR/Jack à l'arrière

Input 1 : combo XLR/Jack channel 1 — première prise à l'arrière. Tu peux aussi utiliser le jack instrument en face avant (Hi-Z) qui prend le pas sur la combo arrière quand un jack est inséré.
Input 2 : combo XLR/Jack channel 2 — idem, jack instrument façade dispo.
Input 3 : combo XLR/Jack channel 3 (mic/line uniquement, pas de Hi-Z).
Input 4 : combo XLR/Jack channel 4 (mic/line uniquement).

Inputs 5 à 12 — ADAT optique

Inputs 5–12 : 8 canaux ADAT Lightpipe via l'entrée optique arrière (à 44.1/48 kHz ; à 96 kHz tu tombes à 4 canaux via SMUX).

Inputs 13–16 — Sources virtuelles
Le SSL 12 présente en réalité 16 entrées au DAW. Les sources additionnelles sont une entrée loopback stéréo qui peut capturer l'audio d'autres applications, et le micro talkback mono intégré qui apparaît comme une entrée stéréo. Sound On Sound

Inputs 13/14 : Loopback stéréo (audio interne du PC, idéal pour streamer/capturer du son d'un navigateur, Traktor, etc.)
Inputs 15/16 : Talkback mic (le micro intégré sur la face supérieure)

**Sorties (8 physiques)

Outputs 1/2 : sortie monitor principale (le gros bouton MONITOR LEVEL agit dessus par défaut) → tes enceintes principales.
Outputs 3/4 : seconde paire d'enceintes ou sorties line additionnelles.
Outputs 5/6 : Headphones A (sortie casque façade gauche).
Outputs 7/8 : Headphones B (sortie casque façade droite).

Configuration recommandée dans Ableton
Dans Préférences → Audio → Input Config, active au minimum :

Mono : 1, 2, 3, 4 (tes préamplis)
Stereo : 1/2, 3/4 (paires utiles), 13/14 (loopback si tu veux capturer le son du PC)

Dans Output Config, active au minimum :

Stereo : 1/2 (monitors), 5/6 (casque A), et 3/4 ou 7/8 selon ton setup
